# HTML Best Practices

## Semantics

- Use semantic elements (`<article>`, `<section>`, `<nav>`) over generic `<div>`
- Headings follow hierarchy — don't skip levels
- Use `<main>` for primary content

## Accessibility

- All images need `alt` attributes
- Form inputs need associated labels
- Use ARIA attributes when native semantics are insufficient

## Structure

- Keep markup clean and minimal
- Avoid unnecessary wrapper elements
- Use `<template>` for client-side rendered content
