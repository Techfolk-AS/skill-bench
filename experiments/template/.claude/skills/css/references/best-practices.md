# CSS Best Practices

## Naming

- Use meaningful class names that describe purpose, not appearance
- Keep selectors shallow — avoid deep nesting
- Group related properties together

## Structure

- Mobile-first approach for media queries
- Use custom properties for theming
- Keep specificity low and consistent

## Responsive

- Prefer relative units (rem, em, %) over fixed pixels
- Use container queries where supported
- Test across breakpoints
